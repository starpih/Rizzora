# AGENTS.md

## Project

Rizzora is an AI virtual male companion product for women in the Middle East, Europe, and North America.

## Required Reading

Before working on Figma, design system, components, or frontend code, read:

- docs/PRODUCT_SPEC.md
- docs/DESIGN_SYSTEM_BRIEF.md
- docs/FIGMA_AUDIT_TASK.md

## Working Rules

- Preserve the existing visual direction.
- Do not redesign screens unless explicitly asked.
- Prefer semantic variables over hardcoded values.
- Keep Figma components ready for frontend implementation.
- Do not add unrelated components.
- Move deprecated elements to 99 Archive instead of deleting them.

## Design Style

Romantic dark luxury, premium, emotional, female-friendly, private, safe, not vulgar.

## First Phase Goal

Audit and optimize the existing Figma Foundations, variables, text styles, effect styles, and component library.
