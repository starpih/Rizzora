# Rizzora Design System Brief

## 1. Design Direction

The design system should support a romantic dark luxury AI companion product.

The visual style should feel:

- Romantic
- Premium
- Emotional
- Female-friendly
- Private
- Safe
- International
- Soft but not childish
- Attractive but not vulgar

## 2. Avoid

Avoid:

- Cheap dating app style
- Live-streaming platform style
- Nightclub style
- Overly sexualized visuals
- Heavy black-red adult style
- Over-gamified UI
- Random neon effects

## 3. Core Visual Language

- Deep purple background
- Warm gold premium accent
- Soft rose emotional accent
- Voice aura glow
- Elegant male portrait cards
- Glass-like dark surfaces
- Rounded cards
- Soft gradients
- Premium membership feeling

## 4. Required Foundations

The Figma design system should include:

- Primitive colors
- Semantic colors
- Typography styles
- Spacing variables
- Radius variables
- Shadow styles
- Glow styles
- Blur styles
- Gradient styles

## 5. Required Components

Core components:

- Button
- Input / Search
- Tag
- Character Card
- Chat Bubble
- Voice Message
- Voice Aura
- Membership Modal
- Bottom Navigation
- State View

## 6. Variable Naming Rules

Use semantic naming:

- color/bg/card
- color/text/primary
- color/action/primary/default
- color/border/default
- color/accent/voice

Avoid unclear names:

- purple1
- button color
- bg new
- color copy
