# Rizzora MVP Product Spec

## 1. Product Positioning

Rizzora is an AI virtual male companion product for women in the Middle East, Europe, and North America.

Users can discover AI male characters, chat with them through text and voice, view character profiles, build emotional relationship progress, and unlock more features through membership.

## 2. Target Users

- Women in the Middle East
- Women in Europe and North America
- Users interested in emotional companionship, romantic voice interaction, and virtual boyfriend experiences

## 3. MVP Goal

The MVP should validate whether users will:

1. Browse AI male characters
2. Enter chat
3. Interact through text and voice
4. Build emotional attachment
5. Subscribe for more features

## 4. P0 Pages

- Home
- For You
- Chat
- Character Profile
- Subscription Modal
- Profile

## 5. P1 Pages

- My Wish
- Create Boyfriend
- Create My Voice
- History
- Pinned
- Voice Settings

## 6. Core User Flow

Home / For You  
↓  
Character Card  
↓  
Chat  
↓  
Character Profile  
↓  
Favorability / Relationship Progress  
↓  
Subscription Modal

## 7. MVP Not Included

- Real AI voice API
- Real payment
- Real login
- Full multilingual system
- Admin system
- Complex creator tools
